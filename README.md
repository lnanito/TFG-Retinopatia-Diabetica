# TFG - Detección de retinopatía diabética

Trabajo de Fin de Grado centrado en el uso de modelos de deep learning para el análisis de imágenes de fondo de ojo.

## Descripción

El objetivo del proyecto es detectar la retinopatía diabética y sus principales signos (microaneurismas, hemorragias y exudados) mediante modelos de redes neuronales convolucionales.

## Estado del proyecto

Actualmente, el proyecto se encuentra en fase de desarrollo. Se ha definido la metodología y la estructura del trabajo. La implementación y el entrenamiento de los modelos están en proceso.

## Estructura del repositorio

- memoria/: documento del TFG
- src/: código del proyecto (pendiente)
- notebooks/: pruebas y experimentos (pendiente)
